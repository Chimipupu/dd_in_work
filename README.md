## 概要

- ESP32でWiFiを通してデータを送受信できること

『H/W』

- ESP32-S3 開発基板
  - TX(送信側)... WiFi MACアドレス(34:85:18:8E:ED:AC)
  - RX(受信側)... WiFi MACアドレス(34:85:18:46:B9:D8)

『開発環境IDE』

- [Arduino IDE ... Ver2.3.6](https://github.com/arduino/arduino-ide/releases/tag/2.3.6)
